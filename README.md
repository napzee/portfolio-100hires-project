# Portfolio Project

## What Tools I Installed
- **Cursor IDE** — AI-powered code editor, downloaded from the official site (cursor.com)
- **Codex** — AI coding assistant extension, found and installed via Cursor's Extensions panel
- **Claude Code** — AI coding assistant, not available through the Extensions marketplace search; enabled instead through Cursor's settings

## What Steps I Completed
1. Downloaded and installed Cursor IDE from the official website
2. Searched for the Codex extension in the Extensions panel and installed it
3. Searched for Claude Code in the Extensions panel but could not find it there
4. Explored Cursor's settings and successfully enabled/activated Claude Code from there
5. Adjusted related settings until both the Codex and Claude AI indicators turned on
6. Created a public repository on GitHub
7. Wrote this README.md documenting the process directly through the GitHub web interface
8. Committed the changes to the repository

## What Issues I Ran Into and How I Solved Them
- **Issue:** Could not find "Claude Code" in the Cursor Extensions marketplace search, even after multiple attempts.
  **Solution:** Could not locate it as a standard extension. Instead, found and activated it through Cursor's settings, where the status indicator turned on successfully.

- **Issue:** "Codex" appeared in the Extensions search, but its AI indicator did not initially show as active.
  **Solution:** After adjusting the relevant settings in Cursor, both the Codex and Claude AI indicators turned on, confirming both tools were successfully connected.

- **Issue:** No clear "Clone Repository" menu was visible in Cursor.
  **Solution:** Edited files directly through the GitHub web interface instead of cloning the repository locally.
